Instrukcja użytkownika – Mini IT Ticket System

Tworzenie zgłoszenia:
1. Użytkownik wprowadza tytuł zgłoszenia
2. Dodaje opis problemu
3. Wybiera kategorię
4. Określa priorytet
5. Zapisuje zgłoszenie

Zarządzanie zgłoszeniami:
- zmiana statusu
- zamykanie zgłoszeń
- analiza zgłoszeń

Raportowanie - system umożliwia generowanie raportów SQL dotyczących:
- liczby zgłoszeń
- kategorii problemów
- statusów zgłoszeń
- priorytetów
