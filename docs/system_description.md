Mini IT Ticket System – System zgłoszeń IT

Opis projektu
Mini IT Ticket System to prosty system wspierający obsługę zgłoszeń IT w organizacji.
System umożliwia:
- rejestrację użytkowników
- tworzenie zgłoszeń
- kategoryzację problemów
- przypisywanie priorytetów
- zarządzanie statusem zgłoszeń
- analizę danych zgłoszeniowych

Zastosowanie
Projekt symuluje realny system helpdesk / service desk wykorzystywany w działach IT.

Funkcjonalności:
- zarządzanie użytkownikami
- obsługa zgłoszeń IT
- raportowanie danych
- analiza zgłoszeń
- workflow zgłoszeń

Technologie:
- MySQL
- SQL
- GitHub
- UML
- Markdown
